
# IP API

A simple API for IP address lookups, ASN info, and hosting/datacenter checks.


## Installation
```bash
git clone https://github.com/mikeyw48/ip-api.git
cd ip-api
npm install
```
    
## Features

- **IP Validation** – Accepts both IPv4 and IPv6 addresses.
- **ASN Lookup** – Finds the Autonomous System Number (ASN) and organization using MaxMind GeoLite2 ASN database.
- **Hosting/Datacenter Check** – Checks if an IP belongs to a known hosting or datacenter provider.
